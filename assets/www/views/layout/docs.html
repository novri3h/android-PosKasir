{{app.name}}

{{app.name}} (Portable) adalah sebuah sistem manajemen dagang dimana data tersimpan secara offline di memori internal HP masing-masing pengguna. Sistem ini memiliki kelebihan yaitu dapat diakses secara cepat karena tidak membutuhkan koneksi internet. Tetapi sistem ini hanya dapat diakses dalam satu jaringan WIFI dan tidak dapat diakses dari tempat yang berbeda lokasi.

Fitur-fitur

Sistem {{app.name}} memiliki fitur-fitur unggulan sebagai berikut :

  1. Dapat diakses melalui komputer dengan koneksi WIFI lokal
  2. Multioperator dimana memiliki hak akses yang berbeda-beda
  3. Dapat memasukan produk barang dan jasa sehingga dapat digunakan untuk bengkel atau jenis usaha jasa yang lain
  4. Terdapat sistem paket diskon untuk memudahkan memberikan promo diskon suatu produk
  5. Terdapat sistem level harga dimana satu produk dapat memiliki harga yang berbeda-beda
  6. Dapat melakukan transaksi tunai maupun kredit dan otomatis tercatat dalam data hutang piutang
  7. Sistem dibuat secara lengkap dan dengan tampilan yang mudah untuk dipahami
  8. Dan banyak fitur menarik lainnya

Cara Pengoperasian

Berikut langkah-langkah untuk menggunakan sistem {{app.name}}.

Mengisi Data Produk
  • Membuat kategori produk

    Kategori produk berfungsi untuk mengelompokan produk berdasarkan jenisnya agar mudah saat pencarian data

  • Input data produk

    Anda dapat input jenis produk barang atau jasa

Mengisi Data Stok
  • Input stok secara manual

    Untuk input stok secara manual dapat melalui menu stok

  • Input stok melalui pembelian

    Pembelian produk dari supplier dapat otomatis mengisi data stok, dapat dilakukan melalui transaksi Pembelian

Mengisi Data Operator
  • Input data operator

    Operator berfungsi untuk karyawan dimana dapat diatur dengan hak akses berbeda-beda

Siap Melakukan Transaksi
  • Login sebagai pemilik toko

    Masuk ke sistem sebagai pemilik toko adalah menggunakan password default 'admin'

  • Login sebagai operator

    Masuk ke sistem sebagai operator adalah menggunakan username dan password sesuai yang dibuat di data operator

Cara Backup Data

Sebaiknya anda sering melakukan backup data untuk menghindari hal yang tidak diinginkan terjadi. Karena perangkat elektronik bisa terjadi error kapan saja baik oleh faktor hardware maupun software

  • Backup ke Dropbox

    Anda dapat menggunakan layanan dropbox untuk melakukan backup secara online, sebelumnya anda harus membuat akun dropbox terlebih dahulu

  • Backup Secara Manual

    Backup secara manual dapat dilakukan dengan cara Copy folder KasirToko yang ada di memori internal HP ke flashdisk atau ke memori external. Dan untuk mengembalikan data anda tinggal replace folder KasirToko dari memori external ke memori internal

Hak akses Operator
  • √ Penjualan

    Operator diizinkan mengakses halaman penjualan

  • √ Retur Penjualan

    Operator diizinkan mengakses halaman retur penjualan

  • √ Pembelian

    Operator diizinkan mengakses halaman pembelian

  • √ Retur Pembelian

    Operator diizinkan mengakses halaman retur pembelian

  • √ Pemasukan

    Operator diizinkan mengakses pencatatan pemasukan untuk toko

  • √ Pengeluaran

    Operator diizinkan mengakses pencatatan pengeluaran untuk toko

  • √ Produk

    Operator diizinkan mengakses data produk, tambah produk baru dan melihat harga

  • √ Stok

    Operator diizinkan mengakses data stok, dapat menambah dan mengurangi stok

  • √ Customer

    Operator diizinkan mengakses data customer

  • √ Supplier

    Operator diizinkan mengakses data supplier

  • √ Edit Harga Penjualan

    Operator diizinkan mengedit harga saat proses penjualan, edit harga dimaksudkan untuk memberikan potongan harga saat tawar menawar

  • √ Paket Diskon

    Operator diizinkan mengakses paket diskon untuk dapat menambah dan mengedit paket diskon

  • √ Level Harga

    Operator diizinkan mengakses level harga untuk dapat menambah dan mengedit level harga

  • √ Edit Tanggal

    Operator diizinkan mengedit tanggal. Ini bagian yang sangat penting dan hanya khusus untuk operator kepercayaan. Karena mengedit tanggal transaksi dapat dimanfaatkan untuk kecurangan data jika operator mengedit tanggal diubah ke waktu yang telah lampau

  • √ Edit data

    Operator diizinkan mengedit data. Sebaiknya diberikan juga hanya ke operator kepercayaan

  • √ Hapus data

    Operator diizinkan menghapus data. Sebaiknya diberikan juga hanya ke operator kepercayaan

  • √ Piutang

    Operator diizinkan mengakses data piutang, dapat menambah dan membayar piutang

  • √ Hutang

    Operator diizinkan mengakses data hutang, dapat menambah dan membayar hutang

  • √ Laporan

    Operator diizinkan mengakses data laporan harian, bulanan, tahunan dan saldo

  • √ Arus uang

    Operator diizinkan mengakses data arus uang
