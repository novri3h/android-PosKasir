Import Produk dari Excel

Untuk Import produk dari file excel anda harus melakukan Export terlebih dahulu. Selanjutnya file excel hasil Export edit dengan mengisi data produk-produk anda, dan isilah semua kolom dengan benar, jangan ada yang kosong, ID produk dapat diisi 1 huruf nanti sistem otomatis membuat ID produk. Format file excel adalah XLS tidak boleh XLSX. Dan nama file excel sebaiknya jangan diganti atau jika diganti jangan ada spasi pada nama file excelnya. Setelah itu tinggal klik browse dibawah ini dan pilih file excel yang telah diisi data produk. Jika produknya sangat banyak sebaiknya lakukan secara bertahap misal per 100 produk, tetapi jangan menghilangkan header keterangan dari tabel pada file excel. Dan proses Import hanya boleh dilakukan pada awal input data produk agar tidak terjadi error atau salah penjumlahan dari laporan keuangan. Pastikan browser menggunakan Firefox dan anda telah membaca manual book dari aplikasi ini.
